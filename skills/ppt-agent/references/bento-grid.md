# 便当网格（Bento Grid）版面方法论

> 内容页布局的核心方法论。设计阶段（生成 SVG）时与 `prompts/04-design-svg.md` 一同加载，指导每页版面。

内容页采用便当网格布局：一种灵活的网格系统，其布局应**由内容本身的需求驱动，而非僵硬的模板**。通过组合不同尺寸的卡片，创造动态且视觉有趣的版面。

## 核心原则
- **灵活性**：卡片数量不固定，可以是 1、2、3、4、5 或更多，取决于如何更好地呈现信息。
- **层级感**：用卡片尺寸建立视觉层级，最重要的信息放在最大的卡片上。
- **留白**：所有卡片之间保持至少 20px 间距。

## 版面组合示例（按 1280×720 画布换算）
- **单一焦点**：一张大卡片覆盖大部分区域（约 w=1200, h=580）。适用于单一、有力的信息或详细图表。
- **两栏布局**
  - *50/50 对称*：两张等宽卡片。
  - *非对称*：一张较宽卡片（约 2/3 宽）放主内容，一张较窄（1/3 宽）放辅助信息、数据或图片。
- **三栏布局**：三张等宽卡片，适合并列比较三项内容。
- **主次结合**：一张居中大卡片，两侧各一张小的垂直卡片。
- **顶部英雄式**：顶部一张宽幅"英雄"卡片，下方 2-4 个较小等宽卡片网格。
- **混合网格（自由度最高）**：自由混合各种尺寸的卡片，例如一个中等方块 + 两个小水平矩形 + 一个垂直矩形，最大程度适应不同内容的需求。
