作为精通信息架构与 SVG 编码的专家，你的任务是将本页内容转化为一张高质量、结构化、具备高级感、简洁感和专业感的 SVG 演示文稿页面。

## 硬性要求
1. **画布**：SVG `viewBox` 必须是 `0 0 1280 720`，所有元素不得溢出画布。
2. **版面**：应用上下文中提供的"便当网格（Bento Grid）"方法论，按本页规划的版面组合落地；卡片间距 ≥20px，用卡片尺寸建立视觉层级。
3. **可编辑性（导入 PPT 的关键）**：
   - 所有文字用 `<text>` 元素承载，**不要转曲、不要转路径**，以便导入 PowerPoint 后仍可改字。
   - 字体用通用中文字体栈：`font-family="'Microsoft YaHei','PingFang SC','Noto Sans CJK SC',sans-serif"`，避免缺字。
   - **不引用任何外部资源**（图片 / 字体 / CSS 的 URL）；所需图标用内联 SVG 路径绘制。
4. **风格**：遵循风格方向 —— {{STYLE}}。配色克制、留白充分、对齐严谨。

## 输出格式
**只输出 SVG 代码本身**，从 `<svg` 到 `</svg>`，不要任何解释文字，不要 markdown 代码围栏。
