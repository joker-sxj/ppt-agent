# 把 SVG 幻灯片用进 PowerPoint

> 交付阶段:把本文要点写进产物目录的 `README.md`,让用户知道怎么用。适用 PowerPoint 2016 及以上。

## 方法 0:直接打开自动生成的 .pptx(默认 · 最省事)

阶段 6 的 `scripts/build_pptx.py` 已把整套打成 `<主题>.pptx`,且**每页都是原生 PowerPoint 对象**:每个矩形、圆、线条、文字都翻译成了真正的形状 / 文本框。
- 双击打开即可放映,也可**直接改字、改色、挪位置、换字体**——无需任何"转换"步骤。
- 仅复杂图标 / 箭头 / 装饰路径合成为一张透明 PNG 叠在最上层(这部分不能逐点编辑;要改就删掉该叠层图片、用形状自行替换)。
- 因为是逐元素翻译,个别文字的换行 / 位置可能与 SVG 预览有细微出入,挑剔场合可在 PPT 里微调。

下面的方法 A / B 是**手工兜底**——只有在没生成 .pptx、或想拿单页 SVG 另作他用时才用。

## 方法 A:逐页插入(最稳)
1. 新建空白演示文稿，设计 → 幻灯片大小 → **宽屏 16:9**。
2. 插入 → 图片 → 来自文件 → 选中 `slides/slide-01.svg`。
3. 选中插入的 SVG → 右键 → **转换为形状**（Convert to Shape）→ 此后文字、色块均可在 PPT 内直接编辑。
4. 拖动铺满整页，逐页重复。

## 方法 B：批量
- 可一次选中多个 SVG 一起插入，再各自"转换为形状"并铺满页面。
- 若只需放映、不需编辑：直接插入 SVG 即可，无需转换（保持矢量清晰）。

## 提示
- SVG 是矢量图，无限放大不糊。
- 转换为形状后若个别字显示异常，在 PPT 里把该文本重选通用中文字体（微软雅黑 / 苹方）即可。
- 先打开产物目录里的 `preview.html` 通览全部页面，确认无误再导入。
